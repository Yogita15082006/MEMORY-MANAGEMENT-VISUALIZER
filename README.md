🧠 Memory Management Visualizer

An interactive web-based simulator that demonstrates how operating systems allocate memory using different allocation strategies. This tool helps visualize dynamic memory allocation and understand fragmentation behavior in OS concepts.

🚀 Features

📦 Implements core memory allocation strategies:

First Fit

Best Fit

Worst Fit

📊 Visual representation of:

Memory blocks

Process allocation

Free spaces

🔄 Real-time updates of memory allocation

🧮 Demonstrates internal and external fragmentation

🛠 Tech Stack

HTML

CSS

JavaScript

🧩 How It Works

User defines memory block sizes.

Processes are allocated using selected allocation strategy.

The algorithm dynamically assigns memory.

Visualization updates to reflect allocated and free memory.

Fragmentation behavior can be observed in real time.
